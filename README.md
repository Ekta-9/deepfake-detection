# Deepfake Detector

A full-stack web application for detecting AI-generated images. Upload any image and the model returns a confidence score along with a GradCAM heatmap that highlights exactly which parts of the image triggered the prediction.

Built as a complete product — user accounts, analysis history, real-time inference, and explainability all included.

---

## Features

- **Image analysis** — upload a JPEG or PNG and get a REAL / UNCERTAIN / FAKE verdict
- **GradCAM overlay** — visual explanation of what the model focused on
- **User accounts** — register, log in, and view your full analysis history
- **Containerised** — runs entirely in Docker, no local dependencies needed

---

## Tech Stack

| | |
|---|---|
| Frontend | HTML, CSS, Vanilla JS, Nginx |

| Model | ResNet50 fine-tuned on the GenImage dataset |

---

## Getting Started

Requires [Docker Desktop](https://www.docker.com/products/docker-desktop).

```bash
git clone <your-repo-url>
git lfs pull
docker compose up --build
```

---

## Training

The model was trained on the [GenImage](https://github.com/GenImage-Dataset/GenImage) dataset covering images generated by 7 different AI systems. Training was done locally on an NVIDIA RTX 3050.
